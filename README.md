# To-Do List App

A simple Spring Boot application to manage daily tasks using HTML, CSS, and MySQL.

## Features
- View all tasks
- Add new tasks
- Auto ID generation with JPA

## Technologies Used
- Java 17
- Spring Boot
- Spring Data JPA
- Thymeleaf
- MySQL
